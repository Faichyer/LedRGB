  • The Arduino control

    The main purpice of this project was to create a projector controlled by two different ways. This projector is made by 3 different color 20W LEDs, an Arduino Uno board with his Ethernet & Bluetooth Shields. Three PWM signals are generated by this former and direcly go to the LEDs, or not ...

    Skip this part

  • Electronic part of the system

    Indeed, the three signals are sent to the LEDs but they pass through MOSFET transistors because their current flow is too low for these LEDs. Also, a microphone is used to detect the beats of a song and makes the LEDs change the colors with the rythms of the music !

    Skip this part

  • Web and Android supervision

    We needed at least two different kind of interfaces to control the LEDs. In one hand, we made a server hosted on the Ethernet Shield of the Arduino and we compute 3 values to make the LEDs change color. In the other hand, we created an Android application which can do the same things but with a lot of optionnal features that you can see next.
